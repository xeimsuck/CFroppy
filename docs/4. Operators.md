<div align="center">
<h3>Binary</h3>
</div>

```froppy
let x = 10;
let y = 5;

x+y; // 15
x-y; // 5
x/y; // 2
x*y; // 50

x>y; // true
x>=y; // true
x<y; // false
x<=y; // false

x+=10; // x==20
y/=10; // y==0.5
```
